# README для приложения "К Редактор"

## Обзор

"К Редактор" — это мощный инструмент, предназначенный для помощи пользователям в редактировании и улучшении текстов. Используя передовые модели ИИ, приложение позволяет улучшать документы в различных стилях, от официальных деловых писем до неформальных разговорных текстов. Интерфейс приложения построен с использованием PyQt6 и включает поддержку мультимедиа для более увлекательного пользовательского опыта.

## Возможности

- **Редактирование текста:** Пользователи могут вводить текст и указывать желаемый стиль редактирования (официальный, научный, разговорный), чтобы получить улучшенные предложения от ИИ.
- **Выбор модели:** Пользователи могут выбирать из нескольких моделей ИИ, таких как:
  - OpenAI GPT-OSS
  - OpenRouter Elephant Alpha
  - NVIDIA Nemotron
- **Настройка API ключа:** Возможность ввода собственного API ключа для подключения к сервисам ИИ.
- **Выбор темы:** Доступны светлая и темная темы для удобства чтения.

## Установка

### Требования

- Python 3.8 или выше
- PyQt6
- Библиотека requests

### Шаги

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-репозиторий/k-redactor.git
   cd k-redactor
   ```

2. Установите необходимые пакеты Python:
   ```bash
   pip install PyQt6 requests
   ```

3. Убедитесь, что у вас есть необходимые мультимедийные файлы:
   - `Main_1.mp4` (для начального экрана загрузки)
   - `Sound.mp3` (для звуковых эффектов)

4. Запустите приложение:
   ```bash
   python main.py
   ```

## Использование

После запуска приложения "К Редактор" вы можете:

- Выбрать стиль редактирования, нажав на соответствующие кнопки.
- Ввести текст и задать запрос к ИИ.
- Нажать на кнопку редактирования, чтобы получить обработанный текст.
- Изменять тему интерфейса по своему желанию.
